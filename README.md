# API Traffic Encoder & Decoder

A robust and lightweight Python Command Line Interface (CLI) tool designed for cybersecurity professionals, developers, and pentesters. It helps to quickly analyze, encode, and decode data formats commonly found in API parameters, authorization headers, and raw network payloads.

##  Features

- **Base64 Encoding/Decoding:** Crucial for analyzing HTTP Basic Authentication headers and encoded API tokens.
- **Hexadecimal (Hex) Conversion:** Perfect for inspecting and manipulating raw network packet payloads.
- **URL Encoding/Decoding:** Essential for handling and parsing secure API query parameters and web requests.

##  How to Run

No external dependencies are required. Just clone and run using native Python 3:

```bash
python api_decoder.py
